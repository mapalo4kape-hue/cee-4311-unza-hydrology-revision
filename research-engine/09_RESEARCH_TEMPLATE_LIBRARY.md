# AgriMine Nexus — Research Template Library (Deliverable 9 / Component 8)

> Index of all reusable authoring templates. Every research artifact is produced from one of these. Each template embeds the mandatory metadata front-matter (D5) and enforces the Citation Standard. Canonical location: [`../templates/`](../templates/).

| # | Template | File | Used for |
|---|----------|------|----------|
| 1 | Research Report | [`research-report.md`](../templates/research-report.md) | ROM-compliant research reports (`01-research`) |
| 2 | Interview Record | [`interview-record.md`](../templates/interview-record.md) | Stakeholder interviews (`04-interviews`) |
| 3 | Survey | [`survey.md`](../templates/survey.md) | Survey instrument + response summary (`05-surveys`) |
| 4 | Evidence Record | [`evidence-record.md`](../templates/evidence-record.md) | Documenting a single source before/after registering it |
| 5 | Stakeholder Profile | [`stakeholder-profile.md`](../templates/stakeholder-profile.md) | Stakeholder register entries |
| 6 | Competitor Profile | [`competitor-profile.md`](../templates/competitor-profile.md) | `09-competitors` |
| 7 | Technology Assessment | [`technology-assessment.md`](../templates/technology-assessment.md) | `07-technology` |
| 8 | Regulatory Assessment | [`regulatory-assessment.md`](../templates/regulatory-assessment.md) | `08-regulations` |
| 9 | Workflow Documentation | [`workflow-documentation.md`](../templates/workflow-documentation.md) | Process/workflow write-ups |
| 10 | Process Map | [`process-map.md`](../templates/process-map.md) | BPMN-style process maps |
| 11 | Risk Register | [`risk-register.md`](../templates/risk-register.md) | Risk registers |
| 12 | Requirement Register | [`requirement-register.md`](../templates/requirement-register.md) | Requirement capture (mirrors RTM CSV) |
| 13 | Feature Register | [`feature-register.md`](../templates/feature-register.md) | Product feature catalogue |
| 14 | Integration Register | [`integration-register.md`](../templates/integration-register.md) | Integration catalogue entries |
| 15 | AI Opportunity Register | [`ai-opportunity-register.md`](../templates/ai-opportunity-register.md) | AI use cases |
| 16 | GIS Opportunity Register | [`gis-opportunity-register.md`](../templates/gis-opportunity-register.md) | GIS opportunities |
| 17 | IoT Opportunity Register | [`iot-opportunity-register.md`](../templates/iot-opportunity-register.md) | IoT opportunities |
| 18 | Activity Log | [`ACTIVITY_RECORD_TEMPLATE.md`](../templates/ACTIVITY_RECORD_TEMPLATE.md) | Activity records (already in use) |
| 19 | Lessons Learned | [`lessons-learned.md`](../templates/lessons-learned.md) | Retrospective per work package |

**Usage:** copy the template into the correct repository folder, rename per the naming convention (D2 §2), fill all mandatory metadata, cite every claim, and link forward/backward per the Traceability Framework (D6).
