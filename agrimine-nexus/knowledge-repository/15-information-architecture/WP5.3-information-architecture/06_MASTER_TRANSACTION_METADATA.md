## Output 11 — Business Metadata Catalogue (business metadata only)
| Metadata attribute | Meaning | Applies to |
|--------------------|---------|-----------|
| Business definition | Plain-language meaning | every information object/entity |
| Business owner / steward | Accountable domain & role | every object (see ownership register) |
| Source stakeholder | Who originates the data | information catalogue |
| Lifecycle state | Creation→deletion stage | information & lifecycle catalogues |
| Confidentiality class | Sensitivity classification | classification register |
| Integrity/Availability importance | Business criticality | information catalogue |
| Retention rule | How long kept & disposition | lifecycle catalogue |
| Geographic & industry scope | Where/which sector applies | information catalogue |
| Provenance / evidence IDs | Traceability to `SRC-`/`F-` | every object |
| Confidence score | Evidence strength | every object |
| Related IDs | Links to WF/STK/REG/CAP/REQ/BO/EVT | every object |
| Reference-data governance | Owner and source list | reference data register |
> Metadata is expressed in **business terms** only. No technical/physical metadata (types, keys, indexes, storage) is defined — deferred to later technical design.
